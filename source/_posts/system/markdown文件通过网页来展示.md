##### 1. 安装相关依赖
> npm install -g i5ting_toc

##### 2. 使用方式
> i5ting_toc 参数 文件名 （参数）

参数：
-  -h, --help             帮助信息
-  -V, --version          显示版本号
- -f, --file [filename]  文件名（默认编译 README.md）
- -o, --open             在默认浏览器中打开
- -v, --verbose          打印详细日志

eg:
> i5ting_toc -f day2.md
